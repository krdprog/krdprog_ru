О чём сайт:

На страницах этого сайта я собираю рабочие рецепты и способы решения возникающих задач, записываю выжимки из учебных материалов, чтобы потом к ним возвращаться и пользоваться в рабочем процессе. Пользуйтесь и вы. Удачи!

Мои интересы:

Большой поклонник GNU/Linux и командной строки. Пишу bash-скрипты, стараюсь автоматизировать рутинные задачи, люблю разбираться в глубинах командной строки.

Для веба я занимаюсь Backend-разработкой на Ruby/Ruby on Rails/Sinatra, верстаю адаптивные сайты и работаю с Vue.js

Занимаюсь развитием Ruby-сообщества в Краснодаре (организую тематические митапы и встречи комьюнити - rubykrd.ru).

Развиваю коллективный блог для Ruby-разработчиков - rubyblog.ru

Развиваю SEO-компанию и сервис для SEO-продвижения - trendseo.ru

Пишу gem-ы для rubygems.org - yandex_xml (парсинг данных из поиска через Яндекс.XML), deboshir - на стадии проектирования (комплексное решение для SEO-оптимизации в Ruby on Rails проектах).

Мой github-профиль: https://github.com/krdprog/

Участвую в Opensource-проектах и пишу конспекты для изучающих программирование с MIT и с CC0 1.0 Universal (CC0 1.0) Public Domain Dedication лицензиями.

Написал конспект лекций для Ruby-школы rubyschool.us, конспект используют в изучении Ruby/Ruby on Rails/Sinatra по статистике Github ежедневно по 40-50 человек. Ссылка на конспекты: https://github.com/krdprog/rubyschool-notes

Пишу telegram-боты и парсеры различной информации (на Ruby).
